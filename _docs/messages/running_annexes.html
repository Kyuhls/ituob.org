Message editor will open on the right. In it you should see two tabs: “Lists Annexed” and “Extra Publications”.

Lists Annexed tab contents are populated automatically, while Extra Publications requires you to add items manually. icon: fas fa-info-circle intent: primary - step: | Switch to the Extra Publications tab, and use the circular plus button to add each publication that normally appears under part B. of Lists Annexed section of each ITU OB issue. notes: - text: |

You need to enter only a short Publication code for each item.

Here are the publication codes to use:
Publication name Publication code
List of ITU Carrier Codes (ITU-T Rec. M.1400 (03/2013)) M1400_ICC
Bureaufax Table (ITU-T Rec. F.170) BUREAUFAX
List of recognized operating agencies (ROAs) ROA

After you click the Add Publication button, your changes will be recorded locally. icon: fas fa-info-circle intent: primary - step: | To remove an erroneously added item, click the red cross button on the right. ---

This guide explains how to add the Lists Annexed message to ITU OB edition using the ITU OB Editor app.

It assumes you have the Issue Editor open for the OB edition you are working on, and that you know the general information about adding messages.
