You need to enter only a short Publication code for each item.

Here are the publication codes to use:
Publication name Publication code
List of ITU Carrier Codes (ITU-T Rec. M.1400 (03/2013)) M1400_ICC
Bureaufax Table (ITU-T Rec. F.170) BUREAUFAX
List of recognized operating agencies (ROAs) ROA
